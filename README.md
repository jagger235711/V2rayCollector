# V2rayCollector


## 项目简介
V2rayCollector 是一个从 Telegram 频道抓取 节点 配置、从订阅链接汇总 节点 的项目。

### 支持的类型

|      节点类型      | 是否支持 |
| :----------------: | :------: |
|       VLESS        |   支持   |
|       VMESS        |   支持   |
|  SS (Shadowsocks)  |   支持   |
| SSR (ShadowsocksR) |   支持   |
|       Trojan       |   支持   |
|     hysteria2      |   支持   |
